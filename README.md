# E-Commerce Sales Analytics Dashboard (Power BI)

## Overview

This project presents an interactive E-Commerce Sales Analytics Dashboard developed using Power BI. The dashboard provides a comprehensive analysis of sales performance, customer behavior, and product-level insights, enabling data-driven decision-making. It focuses on transforming raw data into meaningful visualizations that help stakeholders understand key business metrics such as revenue trends, category performance, and geographical distribution.

## Key Features

* Sales trend analysis over time using line charts
* Category-wise revenue distribution using donut charts
* City-wise sales performance visualization
* KPI card displaying total revenue
* Profit or order trend analysis
* Interactive filters and slicers for dynamic data exploration

## Dataset

The dataset used in this project includes the following attributes:

* Order Date
* Product Name
* Category
* Revenue or Sales
* Profit
* City

The data is sourced from a sample dataset or publicly available datasets such as Kaggle.

## Tools and Technologies

* Power BI for dashboard creation and visualization
* Power Query for data cleaning and transformation
* Data modeling techniques
* Business intelligence and analytics concepts

## Dashboard Insights

* Identifies top-performing product categories
* Highlights sales trends and seasonal variations
* Displays geographical distribution of sales
* Provides insights into customer purchasing behavior
* Supports informed business decision-making

## Use Case

This dashboard is designed for business analysts, sales managers, and e-commerce organizations to monitor performance, optimize sales strategies, and improve overall operational efficiency.
